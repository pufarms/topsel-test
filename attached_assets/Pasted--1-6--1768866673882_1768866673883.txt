회원(사용자) 관리 기능을 구축해줘. 참조 이미지를 참고하되 현재 디자인 기조에 맞춰서 만들어줘.

## 1. 회원 등급 체계 (6단계)

| 등급 | 코드 | 설명 | 자동/수동 |
|------|------|------|----------|
| Guest | GUEST | 비회원, 페이지 권한 설정용 (회원목록에 표시 안함) | - |
| 보류중 | PENDING | 회원가입 완료, 관리자 승인 대기 | 자동 |
| 준회원 | ASSOCIATE | 관리자 승인 완료, Start회원 일부 권한 | 수동(승인) |
| Start회원 | START | 예치금 충전 시 자동 승격, 전월 거래 100만원 미만 | 자동 |
| Driving회원 | DRIVING | 전월 거래 100만원~300만원 | 자동 |
| Top회원 | TOP | 전월 거래 300만원 이상 | 자동 |

등급 자동 조정 규칙 (매월 1일 기준, 나중에 정산/통계 연동 시 구현):
- 전월 거래 합계 < 100만원 → Start회원
- 전월 거래 합계 100만원~300만원 → Driving회원
- 전월 거래 합계 >= 300만원 → Top회원
- 준회원이 예치금 충전 시 → Start회원 자동 승격

## 2. 회원 DB 테이블 (User/Member)

기본 정보:
- id
- username (아이디, unique, 로그인용)
- password (비밀번호)
- grade (등급: PENDING, ASSOCIATE, START, DRIVING, TOP)
- companyName (상호명, 수정불가)
- businessNumber (사업자번호, 수정불가)
- businessAddress (사업자주소)
- representative (대표자명)
- phone (대표연락처)
- managerName (담당자명)
- managerPhone (담당자연락처)
- email (이메일)

재무 정보:
- deposit (예치금, 기본값 0)
- point (포인트, 기본값 0)

상태 정보:
- status (상태: 활성/비활성)
- approvedAt (승인일시)
- approvedBy (승인 관리자 ID)
- createdAt (가입일)
- updatedAt

서류 정보 (나중에 구현):
- contractDocument (회원가입계약서류 URL)
- documentVerified (서류확인 여부)

## 3. 회원관리 페이지 (회원관리 > 사용자 관리)

### 3-1. 상단 대시보드 (항상 표시, 실시간 업데이트)

가로로 배치된 카드 형태:
┌─────────┬─────────┬─────────┬─────────┬─────────┬─────────┐
│ 전체회원 │ 보류중   │ 준회원   │ Start   │ Driving │ Top     │
│   00명  │  00명   │   00명  │   00명  │   00명  │   00명  │
└─────────┴─────────┴─────────┴─────────┴─────────┴─────────┘

각 카드 클릭 시 해당 등급으로 필터링

### 3-2. 검색 및 필터 섹션

검색 필드:
- 상호명(사용자명) 검색
- 아이디 검색
- 등급 선택 (전체/보류중/준회원/Start/Driving/Top)
- 정렬 (가입일 최신순, 가입일 오래된순, 상호명순, 예치금순)
- [검색] 버튼
- [초기화] 버튼

### 3-3. 선택 회원 일괄 수정 섹션

체크한 회원에게 일괄 적용:
- 등급 변경 (드롭다운)
- 예치금 조정 (입력, 예: 10000 또는 -5000)
- 포인트 조정 (입력, 예: 1000 또는 -500)
- 메모 추가 (기존 메모에 추가됨)
- [선택 회원 저장] 버튼

### 3-4. 회원 목록 테이블

컬럼:
- ☑ (체크박스)
- 상호명(사용자명)
- 아이디
- 등급 (뱃지 스타일, 색상 구분)
- 예치금 (천원 단위 콤마)
- 포인트 (천원 단위 콤마)
- 연락처 (대표연락처)
- 가입일 (YYYY-MM-DD)
- 사업자등록증 [보기] 버튼 (나중에 구현)
- 통신판매업 [보기] 버튼 (나중에 구현)

행 클릭 시 회원 상세 페이지로 이동

페이지네이션: 총 00명 (1/1 페이지)

엑셀 다운로드 버튼 (.xlsx 형식)

### 3-5. 등급별 뱃지 색상

- 보류중: 노란색/주황색 배경
- 준회원: 회색 배경
- Start: 파란색 배경
- Driving: 초록색 배경
- Top: 보라색/금색 배경

## 4. 회원 상세/수정 페이지

회원 목록에서 클릭 시 이동

### 수정 불가 항목 (읽기전용, 회색 배경):
- 아이디
- 사업자번호
- 상호명

### 수정 가능 항목:
- 등급 (드롭다운)
- 대표자명
- 사업자주소
- 대표연락처
- 담당자명
- 담당자연락처
- 이메일
- 예치금 조정 (+/- 입력)
- 포인트 조정 (+/- 입력)
- 상태 (활성/비활성)
- 메모

### 특수 기능:
- [비밀번호 초기화] 버튼 - 클릭 시 확인 후 임시 비밀번호로 초기화
- [승인] 버튼 - 보류중 회원만 표시, 클릭 시 준회원으로 승격
- [저장] 버튼
- [목록으로] 버튼

### 수정 이력:
- 하단에 수정 이력 표시 (누가, 언제, 무엇을 변경했는지)

## 5. 접근 권한

- 최고관리자: 모든 기능 접근 가능
- 관리자: 권한 설정에 따라 접근 (members.user 권한)

## 6. 반응형 디자인

- 데스크톱: 테이블 전체 표시
- 태블릿: 일부 컬럼 숨김 (사업자등록증, 통신판매업)
- 모바일: 카드 형태로 변환, 검색/필터 접기 가능

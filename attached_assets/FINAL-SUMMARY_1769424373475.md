## 🚀 Replit 사용 방법 (3단계)

### **1단계: 프로젝트 설정**
```bash
# Replit Console에서
unzip topsel-design-system-complete.zip

# 파일 구조 확인
ls -la
```

### **2단계: 모든 HTML에 CSS 링크**
```html
<head>
  <link rel="stylesheet" href="design-system-global.css">
</head>
```

### **3단계: 새 페이지 작성 프롬프트**
```
"design-system-responsive.json과 REPLIT-STRICT-GUIDELINES.md를 
100% 준수해서 [페이지명]을 만들어줘.

필수:
- design-system-global.css 링크
- 섹션 배경 White ↔ Navy 교차
- 정의된 클래스만 사용 (.h1-hero, .btn-primary 등)
- 375px, 768px, 1024px 테스트
- 최종 체크리스트 확인"
```

## ⚠️ Replit이 절대 어겨서는 안 되는 5가지

### 1. **임의의 색상 사용 금지**
```css
/* ❌ 금지 */
color: #FF5733;

/* ✅ 허용 */
color: var(--primary);
```

### 2. **임의의 폰트 크기 금지**
```html
<!-- ❌ 금지 -->
<h1 style="font-size: 60px;">제목</h1>

<!-- ✅ 허용 -->
<h1 class="h1-hero">제목</h1>
```

### 3. **섹션 배경 연속 금지**
```html
<!-- ❌ 금지 -->
<section class="section-dark">...</section>
<section class="section-dark">...</section>

<!-- ✅ 허용 -->
<section class="section-dark">...</section>
<section class="section-light">...</section>
```

### 4. **Desktop-first 금지**
```css
/* ❌ 금지 */
@media (max-width: 768px) { ... }

/* ✅ 허용 */
@media (min-width: 768px) { ... }
```

### 5. **디자인 시스템 무시 금지**
- 새로운 버튼 스타일 만들기 ❌
- 새로운 카드 스타일 만들기 ❌
- 정의되지 않은 요소는 **사용자에게 확인** ✅

## 🎨 핵심 디자인 규칙 요약

### **색상 팔레트**
```
Primary: #5D7AF2 (파란색)
Navy: #111827 (진한 남색 배경)
Accent Green: #10B981 (통계 숫자)
Accent Orange: #FF6B00 (강조)
Accent Cyan: #22D3EE (하이라이트)
```

### **타이포그래피**
```
h1 (Hero): 32px → 56px (반응형)
h2 (Section): 24px → 36px
h3 (Card): 18px → 22px
Stat Number: 32px → 48px
Body: 14px → 16px
```

### **Breakpoints**
```
xs:  375px  (모바일)
sm:  640px  (큰 모바일)
md:  768px  (태블릿)
lg:  1024px (데스크톱)
xl:  1280px (큰 데스크톱)
```

### **섹션 패턴**
```
Header (Light) → Hero (Dark) → Features (Light) → 
Content (Dark) → Stats (Light) → Testimonial (Dark) → 
CTA (Primary) → Footer (Dark)
```

```

1) 기존 스타일 중 [A], [B], [C] 사용
```

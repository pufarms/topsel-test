# 재고관리 탭 추가 (1단계)

## 📋 개요

관리자 페이지에 **재고관리** 탭을 새로 추가하고, 6개의 세부 메뉴를 생성해주세요.
각 세부 페이지는 일단 **"준비 중"** 표시만 해주세요.

---

## 🗂️ 메뉴 구조

```
관리자 페이지
│
├── 상품관리 (기존 - 변경 없음)
│   ├── 카테고리 관리
│   ├── 상품등록 (공급가 계산)
│   ├── 차주 예상공급가 상품
│   ├── 현재 공급가 상품
│   └── 공급 중지 상품
│
└── 재고관리 ← 새로 추가!
    ├── 재료 카테고리 관리
    ├── 재료 관리
    ├── 상품 매핑
    ├── 재고 현황
    ├── 입고 관리
    └── 재고 이력
```

---

## 🔒 접근 권한

**재고관리 전체 메뉴는 관리자 전용입니다.**

| 사용자 | 접근 |
|--------|:----:|
| 관리자 | ✅ 접근 가능 |
| 회원 | ❌ 접근 불가 |

- 회원이 URL 직접 입력해도 접근 불가 (리다이렉트 또는 403)
- 회원에게는 재고관리 메뉴 자체가 보이지 않음

---

## 📑 세부 메뉴 설명

| 순번 | 메뉴명 | 경로 (예시) | 설명 |
|:----:|--------|------------|------|
| 1 | 재료 카테고리 관리 | /admin/inventory/categories | 원재료/반재료/부재료 카테고리 관리 |
| 2 | 재료 관리 | /admin/inventory/materials | 원재료, 반재료, 부재료 등록/수정 |
| 3 | 상품 매핑 | /admin/inventory/mapping | 상품 ↔ 재료 매핑 설정 |
| 4 | 재고 현황 | /admin/inventory/stock | 현재 재고 조회 |
| 5 | 입고 관리 | /admin/inventory/receiving | 재료 입고 등록 |
| 6 | 재고 이력 | /admin/inventory/history | 입출고 내역 조회 |

## ✅ 체크리스트

### 메뉴 구조
- [ ] 사이드바에 "재고관리" 메뉴 추가
- [ ] 재고관리 하위 6개 메뉴 추가
- [ ] 메뉴 클릭 시 해당 페이지로 이동
- [ ] 메뉴 접기/펼치기 기능 (상품관리와 동일)

### 페이지
- [ ] 재료 카테고리 관리 페이지 (준비 중)
- [ ] 재료 관리 페이지 (준비 중)
- [ ] 상품 매핑 페이지 (준비 중)
- [ ] 재고 현황 페이지 (준비 중)
- [ ] 입고 관리 페이지 (준비 중)
- [ ] 재고 이력 페이지 (준비 중)

### 접근 권한
- [ ] 재고관리 전체 관리자 전용 설정
- [ ] 회원에게 재고관리 메뉴 숨김
- [ ] 회원이 URL 직접 입력 시 접근 불가

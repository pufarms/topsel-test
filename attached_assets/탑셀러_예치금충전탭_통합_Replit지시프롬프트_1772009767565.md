## PART 1 — deposit-tab.tsx 컴포넌트 신규 생성

`client/src/pages/member/deposit-tab.tsx` 파일을 새로 만들어줘.

### 데이터 로딩
페이지 진입 시 GET /api/member/profile 로 아래 데이터를 불러와줘:
- memberName — 회원명 (입금자 확인용 오렌지 박스에 크게 표시)
- grade — 회원 등급
- isPostpaid — 후불결제 여부 (boolean)
- balance — 예치금 잔액
- pointer — 포인터 잔액

이번 달 / 지난 달 매입 총액은 기존 대시보드에서 사용 중인 API 재활용해줘.

### 화면 구성 (첨부 HTML 참고)

**상단 히어로 배너 (네이비 그라데이션)**
- "👋 환영합니다!" + "예치금 충전 안내" 타이틀
- 안내 문구
- [이용가이드 보기] [1:1 문의하기] 버튼
- 회원 현황 카드 3개: 회원 등급 / 지난 달 매입 총액 / 이번 달 매입 총액

**후불결제 회원 전용 (isPostpaid=true 시만 표시)**
- 히어로 배너 하단: 보라색 "🔔 후불결제 회원" 배너
- 콘텐츠 최상단: 후불결제 정산 안내 카드 (보라색 테마, 3단계 설명)

**섹션 1 — 핵심 안내 카드**
- 파란색 tip 박스
- isPostpaid=true 시 "후불결제 회원은 동일 계좌로 정산도 자동 처리" 문구 추가
- 농협 계좌 박스 (네이비 그라데이션):
  - 계좌번호: 301-0339-1231-81
  - 예금주: 현 농업회사법인 주식회사
  - [📋 계좌번호 복사] 버튼 (클릭 시 클립보드 복사 + 토스트 메시지)

**섹션 2 — 입금 방법 카드 (3단계)**

STEP 1 — 입금자명 확인 (가장 중요!)
- ★ 회원명 강조 박스 (오렌지 테두리):
  - "반드시 아래 회원명으로 입금해 주세요!" 헤더
  - memberName 값을 크고 굵게 표시
  - 안내 문구 4줄:
    1. 자동 처리 시스템이 이 회원명으로 입금자를 매칭합니다
    2. 사업자 대표자명과 다를 수 있습니다
    3. 띄어쓰기도 동일하게 입력해 주세요
    4. 회원명 변경은 회원정보 페이지에서 수정 가능 (클릭 시 /dashboard?tab=member-info 이동)
- 빨간 경고 박스: "회원명과 입금자명이 다를 경우 자동 처리 안 됨"

STEP 2 — 지정 계좌로 입금

STEP 3 — 자동 처리 완료
- 보통 10분 이내 / 최대 30분~1시간 / 24시간 운영
- 1시간 경과 후 미처리 시 고객센터 문의

**섹션 3 — 환불 및 세금계산서 안내 카드**
- 경고: "예치금 입금 시 세금계산서 발행 안 됨"
- 환불 안내 / 환불 신청 방법 / 세금계산서 발행 안내

**하단 CTA 박스**
- "문의사항이 있으신가요?" + [고객센터 문의하기] 버튼

## PART 2 — dashboard.tsx에 deposit 탭 연결

기존 member-info-tab.tsx가 연결된 방식을 그대로 참고해서
deposit-tab.tsx를 `?tab=deposit` 에 연결해줘.
기존 탭 연결 코드는 절대 수정하지 말 것.

---

## PART 3 — 기존 4곳 버튼 링크 연결

아래 4곳 버튼 클릭 시 `/dashboard?tab=deposit` 으로 이동하도록 연결해줘.
버튼 텍스트, 색상, 스타일은 절대 변경하지 말 것.

**① 회원정보 탭 (member-info-tab.tsx)**
- 오렌지색 [예치금 충전] 버튼
- useNavigate() 로 `/dashboard?tab=deposit` 이동

**② 신규주문등록 탭**
- 잔액 현황 우측 상단 [충전하기] 버튼
- useNavigate() 로 `/dashboard?tab=deposit` 이동

**③ 정산 관리 탭**
- 내 잔액 현황 하단 [예치금 충전 안내] 박스 전체
- cursor: pointer 추가 후 클릭 시 `/dashboard?tab=deposit` 이동

**④ 마이페이지 대시보드 메인**
- 초록색 [예치금 충전 안내] 버튼
- useNavigate() 로 `/dashboard?tab=deposit` 이동
